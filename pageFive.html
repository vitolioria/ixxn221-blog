CSS Box grids

This section was learnt throughout week 5 (CSS grids pt1) and week 8 (CSS grids pt2).

The CSS grid layout module is a helpful and useful coding tool for organizing and arranging content on a webpage. It's like using tables to structure your content into columns and rows, but it's much more versatile. With CSS grid, we are able to create complex layouts, defining how different parts of your webpage should be sized, positioned, and layered in relation to each other. It's not limited to a rigid table structure, and it allows elements within a grid to overlap and stack on top of each other, much like when you position elements using CSS.

We can use different properties to help create grids. These can include (and are not limited to):
  • grid-template-columns: defines the line names and track sizing functions of the grid columns.
  • grid-template-rows: defines the line names and track sizing functions of the grid rows.
  • grid-template-areas: specifies named grid areas, establishing the cells in the grid and assigning them names.
